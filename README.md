# モンくるシミュレータ

Godot 4.6 で動作する「モンスターくるくる」シミュレータです。

## 動作環境

- **Godot Engine 4.6**
- 画面サイズ: **640x480（固定）**
- メインシーン: `res://scenes/Main/Main.tscn`

## 起動方法

### Godot エディタから起動

1. Godot 4.6 でこのリポジトリを開く
2. `F5` で実行

### CLI から起動（`godot4` コマンドを使う場合）

```bash
godot4 --path .
```

## メインメニュー

- 練習モード
- 2P対戦
- VS COM
- オンライン対戦
- キーコンフィグ
- 音量設定
- ゲームを終了

## モード概要

### 練習モード

- 名前・ステージ・ステータス（速度/パワー/くる数など）を設定して開始
- アイテム 3 スロットを設定可能
- リプレイ保存/読込に対応
- 対戦中にチャット入力を開いてメッセージ表示可能

### 2P対戦

- 1P/2P それぞれで、名前・キャラ・くる・アイテムを設定
- ステージ選択可能
- リプレイ保存/読込に対応

### VS COM

- 1P と COM の対戦設定（名前・キャラ・くる・アイテム・ステージ）
- リプレイ保存/読込に対応
- 対戦中にチャット入力を開いてメッセージ表示可能

### オンライン対戦

1. 名前・接続先 IP・キャラ・くる・ステージ・アイテムを設定
2. 片側が「サーバーとして開始」、もう片側が「クライアントとして接続」
3. 接続後、双方が「準備完了」を押すと開始

- 通信方式: ENet（Godot 標準）
- 使用ポート: **`9999`（固定）**
- ステージは双方の選択をもとに開始時に抽選
- 対戦中のチャット送受信に対応
- オンラインリプレイ保存/読込に対応

## ステージ

現在のステージ定義は `data/maps/` 配下の 4 つです。

- クリーム村
- クリスマス村
- サイコロ村
- 海底の村

## キーコンフィグ

キーはゲーム内の「キーコンフィグ」で変更できます。

### 区分

- 練習・オンライン用
- VS 1P 用
- VS 2P 用

### 既定キー

#### 練習・オンライン

- 移動: `→ / ← / ↓ / ↑`
- ショット: `Space`
- アイテム1/2/3: `A / S / D`

#### VS 1P

- 移動: `H / F / G / T`
- ショット: `C`
- アイテム1/2/3: `A / S / D`

#### VS 2P

- 移動: `→ / ← / ↓ / ↑`
- ショット: `\`
- アイテム1/2/3: `L / ; / :`

## 設定・保存データ

### 設定ファイル

- `user://settings.json`
- 主な保存対象:
  - 練習メニュー設定
  - 2P 対戦メニュー設定
  - VS COM メニュー設定
  - オンラインメニュー設定（名前/IP/キャラ/くる/ステージ/アイテム）
  - キー設定
  - 音量設定（BGM/効果音）

### リプレイファイル

- 保存先: `user://replays/`
- スロット: `00`〜`99`
- 拡張子: `.dat`
- ファイル名:
  - 練習: `placXX.dat`
  - 2P対戦: `vsXX.dat`
  - VS COM: `vscomXX.dat`
  - オンライン: `onlineXX.dat`

## トラブルシュート

### オンライン接続できない

- サーバー側が先に起動しているか確認
- 接続先 IP アドレスに誤りがないか確認
- ファイアウォールで **TCP/UDP `9999`** が遮断されていないか確認

### キー設定や音量が反映されない

- ゲーム再起動後に反映されるか確認
- 必要に応じて `user://settings.json` を削除して初期化

### リプレイが見つからない

- `user://replays/` 配下にファイルがあるか確認
- 保存モードと読み込みモード（練習 / 2P / VS COM / オンライン）が一致しているか確認
