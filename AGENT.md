# AI Agent Instructions for this Repository

## Pull Request Guidelines
- プルリクエスト（PR）のタイトルおよび説明文は、常に**日本語**で作成してください。
- タイトルは簡潔かつ明快にし、必要に応じてプレフィックス（feat:, fix:, refactor:, docs: など）を付与してください。
- 説明文には以下のセクションを含めてください：
  - **概要**: 今回の変更内容の要約
  - **変更理由**: なぜこの修正・機能追加が必要になったか
  - **主な変更点**: 具体的な修正箇所の箇条書き
- 文体は、エンジニアリングの現場で自然な「です・ます調」を使用してください。

## General Communication
- コードの解説や提案も、特別な指示がない限り日本語で行ってください。
